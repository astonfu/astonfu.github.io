# 改变 html
在有关获得的方法里，第一个参数是要获得的内容，如果有第二个参数，那么就是设置了。比如：

```js
// 获得 name 的值
$('#title').attr('name')
// 设置 name 的值
$('#title').attr('name', 'Jack')
```

主要方法：

- attr('name')
- css('color')
- addClass('klass')
- removeClass('klass')
- toggleClass('klass')
- data('xxxx') 获得 data-xxxx 定义的数据
- html()
- val()
