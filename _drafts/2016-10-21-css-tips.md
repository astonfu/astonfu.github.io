## position

- static
- relative
- absolute
- fixed

static 是position的默认属性，不会被“positioned”，其它的值可以被“positioned”。

relative 是基于上一个元素来相对定位的。

absolute 会基于不是static的父外框来确定自己的位置，直到是body，而fixed是整个页面的。

## float

float可以实现文字环绕。

## clear
如果前面有float，则去下面，不重叠。

```css
.after-box {
  clear: left;
}
```

左边有float时，就在下面显示。

## overflow
清除浮动，能让大框完全包着浮动的内容。

```css
.clearfix {
  overflow: auto;
}
```

## 响应式布局

```css
@media screen and (min-width:600px) {
  nav {
    float: left;
    width: 25%;
  }
  section {
    margin-left: 25%;
  }
}
@media screen and (max-width:599px) {
  nav li {
    display: inline;
  }
}
```

当页面宽度小于600px时，nav就会显示在上面。

## column

```css
.three-column {

  -moz-column-gap: 1em;
  -webkit-column-count: 3;
  -webkit-column-gap: 1em;
  column-count: 3;
  column-gap: 1em;
}
```
