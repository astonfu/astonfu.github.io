## Anchor —— 锚点

定义一个锚点：

其实就是一个带 id 的 link，带上 name 是为了兼容。

```html
<a id="bookmark-1" name="bookmark-1"></a>
```

跳转到锚点，在现有的链接后加 #id 即可，如上的：

```html
<a href="#bookmark-1">Bookmark 1</a>
```

当然也可以在 url 后直接跟 #id。
